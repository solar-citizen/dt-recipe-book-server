# Installation

```bash
cd server

npm i
```

## Linting and code styling

### If you use WebStorm

1. Go to .prettierrc and accept using code scheme in the notification.
2. Go to settings, search for "prettier"
3. Check automatic prettier configuration
4. Check "Run on save"
5. Click "all actions on save..." link
6. Check:
    - Reformat code
    - Optimize imports
    - Rearrange code
